- **Aliases**: `pcm`

## Arguments

### `<COMMIT_MSG_FILE>`

The path to the file that contains the commit message so far

### `[SOURCE]`

The source of the commit message (e.g., "message", "template", "merge")

### `[SHA]`

The SHA of the commit being amended (if applicable)
