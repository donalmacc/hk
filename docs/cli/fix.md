# `hk fix`

- **Usage**: `hk fix [FLAGS]`
- **Aliases**: `f`

Fixes code

## Flags

### `-a --all`

Run on all files instead of just staged files

### `--linter... <LINTER>`

Run on specific linter(s)

### `--from-ref <FROM_REF>`

Start reference for fixing files (requires --to-ref)

### `--to-ref <TO_REF>`

End reference for fixing files (requires --from-ref)
