# `hk`

**Usage**: `hk [FLAGS] <SUBCOMMAND>`

**Version**: 0.2.2

- **Usage**: `hk [FLAGS] <SUBCOMMAND>`

## Global Flags

### `-j --jobs <JOBS>`

Number of jobs to run in parallel

### `-p --profile... <PROFILE>`

Profiles to enable/disable prefix with ! to disable e.g. --profile slow --profile !fast

### `-v --verbose...`

Enables verbose output

### `-q --quiet`

Suppresses output

### `--silent`

Suppresses all output
